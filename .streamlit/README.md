# 🩺 NIRAMOY-24

**আপনার ২৪/৭ বাংলা ভাষার স্বাস্থ্য সহায়ক**

NIRAMOY-24 একটি AI-চালিত হেলথ কোপাইলট, যা সাধারণ স্বাস্থ্য বিষয়ক প্রশ্নের উত্তর দেয় সহজ বাংলায়, এবং প্রেসক্রিপশনের ছবি পড়ে বুঝিয়ে দেয় — নিরাপদ, দায়িত্বশীল এবং তথ্যভিত্তিক উপায়ে।

> ⚠️ এটি ডাক্তারের বিকল্প নয়। এটি শুধুমাত্র সাধারণ তথ্য ও প্রাথমিক দিকনির্দেশনার জন্য।

## ✨ ফিচারসমূহ

- **💬 বাংলা স্বাস্থ্য চ্যাটবট** — যেকোনো স্বাস্থ্য বিষয়ক প্রশ্নের উত্তর সম্ভাব্য কারণ, করণীয়, এবং জরুরি সতর্কতা সহ
- **🚨 সেফটি গার্ডরেইল** — কোনো diagnosis বা ওষুধের dosage সুপারিশ করে না; জরুরি উপসর্গে সরাসরি হাসপাতালে যেতে বলে
- **📄 প্রেসক্রিপশন রিডার** — প্রেসক্রিপশনের ছবি আপলোড করলে Gemini Vision দিয়ে পড়ে সহজ বাংলায় ওষুধের তালিকা দেখায়
- **🎨 কাস্টম বাংলা UI** — Hind Siliguri ফন্ট ও নিজস্ব ব্র্যান্ডিং সহ ডিজাইন করা

## 🚀 লোকালি চালানোর নিয়ম

```bash
# রিপোজিটরি ক্লোন করো
git clone https://github.com/shahariar-ai/niramoy-24.git
cd niramoy-24

# ভার্চুয়াল এনভায়রনমেন্ট বানাও
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate   # Mac/Linux

# লাইব্রেরি ইনস্টল করো
pip install -r requirements.txt

# .env ফাইল বানাও এবং Gemini API key বসাও
# GEMINI_API_KEY=your_key_here

# অ্যাপ চালাও
streamlit run app.py
```
